# usaha-id pack

**Origin:** made · See project [[usaha-id]].

A Claude Code plugin bundle of workflow skills:
- `usaha-router` (entity resolver) · `usaha-onboard` (entity profiles/connectors)
- `gaji-bpjs` (payroll: BPJS + PPh 21 TER) · `tutup-bulan` (month-end close)
- `pajak-prep` (SPT Masa/Tahunan prep) · `rkab-pnbp` (mining royalty / RKAB)
- `brief-senin` (one-screen cross-entity Monday brief)

Defers to: `indonesian-contract` / `contract-reviewer` / `indonesian-law`,
`glu` (brand), `portfolio` (equities), `mmi` (HPM math), `decide` (strategic forks).
