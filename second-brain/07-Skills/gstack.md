# gstack

**Origin:** made · **v1.1.0** · allowed-tools: Bash, Read, AskUserQuestion.

with elements, verify state, diff before/after, take annotated screenshots, test
responsive layouts / forms / uploads / dialogs, capture bug evidence. Triggers:
"browse this page", "take a screenshot", "navigate to url", "inspect the page".
