What it composes

It sequences the specialists — it doesn't replace them

The workspace already has the taste. The Head of Design is the single owner that plans the work, dispatches the right skill or sub-agent, owns the seams between their outputs, and gates the result.

Read-only reviewer — the quality gate. Run it before declaring done; act on what it finds.
The composition workflow

Cheap thinking up front makes the expensive work land

Intake — turn the brief into a thesis
Restate the job in one line: who it's for, the single feeling it must provoke, the one thing it has to get right. Ask once (batched) only if a genuinely blocking choice remains — otherwise pick the default and move.
Reference — never start from blank taste
Dispatch reference-scout for the domain and adapt the references' structure and motion toward the workspace's bold sports identity — never clone visuals.
Compose — write the plan before any pixels
A vetoable plan: the spine (sections/screens in order), the system (palette, type, motion, the one signature move), and an assignment table — who owns what, in what sequence.
Dispatch — parallel where independent, serial where dependent
Launch independent specialists concurrently; serialize true dependencies (brand tokens before the sections that consume them). Own the seams — shared variables, one type scale, one motion vocabulary.
Integrate & critique — close the loop
Assemble, run design-critic, fix, and converge. One critique pass isn't the job — clearing the bar is. Then ship.
The house bar

Non-negotiables every output is judged against

A specialist's work that violates these isn't done. It gets sent back — or fixed in place.

No purple — ever Geist / Inter, deliberate scale Airy canvas + mint glass Phosphor icons, thin weight One disciplined accent Expo-out, reduced-motion safe WCAG AA · focus rings · keyboard
Install

One file. It lives with the other specialists.

Sub-agents live in ~/.claude/agents/. Drop the definition in, then invoke it on any open-ended, multi-discipline brief.
